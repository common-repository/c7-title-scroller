Scroll page titles of your blog left to right or right to left in the browser.

== Description ==

Does your blog or website has long page titles as part of SEO and you would like your users to read all of them ? or simply want to grab attention for your blog window in the users computer, with an animated page title..? Then you might want to consider this plugin. This plugin makes the web page title scroll, based on the options you setup in the wordpress administrator.


== Installation ==

1.Click Add New menu in Plugins menu of wordpress administrator 
2.Upload 'c7titlescroller' install the plugin
3.Activate the plugin through the 'Plugins' menu in WordPress
4.Click on 'C7 Title Scroller' in Manage options section of wordpress administrator
5.Select and Update the settings in the screen or leave default values
